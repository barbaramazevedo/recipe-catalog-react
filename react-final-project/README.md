# 🍽️ Recipe Catalog App

Aplicação desenvolvida em React para consulta e gerenciamento de receitas culinárias.

## 📖 Sobre o Projeto

O Recipe Catalog App permite que usuários explorem receitas, pesquisem pratos, visualizem ingredientes e modo de preparo, além de salvar suas receitas favoritas.

O projeto foi desenvolvido como atividade final do módulo de React, aplicando conceitos de:

* React Components
* React Router
* Context API
* useReducer
* Consumo de APIs
* Styled Components
* Responsividade

## ▶️ Como Executar

Clone o repositório:

```bash
git clone <url-do-repositorio>
```

Entre na pasta:

```bash
cd recipe-catalog
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```
